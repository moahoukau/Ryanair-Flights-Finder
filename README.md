# Ryanair-Flights-Finder

## Project Description

Ryanair Flights Finder is a desktop application for searching low-cost flights with flexible travel parameters.  
The application is currently focused on direct Ryanair routes and is designed to help users quickly find affordable travel options for spontaneous trips, student travel, short getaways, or budget vacations.

At the current stage, the application allows the user to manually enter:
- the departure airport code
- the departure date range
- the length of stay

The user can then search either:
- all available direct destinations
- seaside destinations only

Based on these inputs, the program searches through all possible direct Ryanair routes from the selected departure airport and displays the available options sorted by ascending price.

## Current Functionality

The current version of the application:
- supports manual input of airport IATA codes
- supports flexible input of travel dates
- supports flexible trip duration
- searches only direct routes
- works only with Ryanair flights
- displays results sorted from the lowest price to the highest
- allows convenient review of low-cost travel options

The application is intended to simplify the process of finding cheap flights when the exact departure date or trip length is not fixed.

## Project Status

This project is still under development.

At the current stage, the application is focused on the core flight-search functionality.  

## Planned Improvements

The following improvements are definitely planned for future versions:
- improved graphical user interface
- airport selection from predefined lists instead of manual code input
- travel collections for easier browsing of interesting trip options

Additional features that may be implemented in the future, depending on available resources and suitable data sources:
- support for additional airlines
- accommodation search
- broader trip-planning functionality
If such features are implemented, they may either be integrated into this application or developed as part of a separate project with expanded functionality.
